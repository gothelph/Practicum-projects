# Портфолио проектов Frontend-разработчика

Портфолио проектов, выполненных в рамках обучения на курсе «Frontend-разработчик» и в ходе самостоятельной практики.

Каждый проект представляет собой реализацию пользовательского интерфейса или веб-приложения с использованием современных технологий frontend-разработки.

Ключевые технологии: JavaScript, TypeScript, React, Redux Toolkit, HTML, CSS, Flexbox, Grid, REST API, Git.

## КАК ИСПОЛЬЗОВАТЬ

1. Клонировать репозиторий:

```
git clone https://github.com/gothelph/Practicum-projects.git
```

2. Перейти в папку нужного проекта

3. Установить зависимости (если есть):

```
npm install
```

4. Запустить проект:

```
npm start
```

или открыть `index.html` в браузере
