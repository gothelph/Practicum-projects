Это моё портфолио, где собраны проекты, выполненные во время обучения на курсе **Frontend-разработчик** в Яндекс.Практикуме и личные pet-проекты.  

## Проекты

### 1. Вёрстка. Семантика и лейаут «Оно тебе надо»
- Сверстан по макету, адаптивен под все экраны  
- **Технологии:** HTML, CSS, BEM, Flexbox, Grid  
- **Code:** (https://github.com/gothelph/ono-tebe-nado?tab=readme-ov-file)  

### 2. Проектная работа «Посмотри в окно»
- Веб-приложение для просмотра видео с интерактивными карточками: выбор видео, подгрузка новых, поиск и обработка ошибок.  
**Технологии:** HTML, CSS, Flexbox, Grid, JavaScript  
- **Code:** (https://github.com/gothelph/posmotri_v_okno) 

### 3. Проектаная работа «Сложно сосредоточиться»
Адаптивный сайт с двумя темами: светлой и тёмной, с автоматическим переключением по системным настройкам пользователя и ручным выбором темы.  
**Функциональность:**  
- Адаптивная верстка Mobile First;  
- Две темы оформления с CSS-переменными;  
- Семантическая разметка и поддержка скринридеров.  
**Технологии:** HTML, CSS, Flexbox, Grid, Media Queries  
- **Code:** (https://github.com/gothelph/slozhno-sosredotochitsya)
